# Rinha de Backend 2026 - Rust IVF

Submissão experimental paralela em Rust para a Rinha de Backend 2026.

O runtime HTTP e a classificação são implementados em Rust. A primeira versão usa o índice IVF binário já validado na submissão C++ como artefato de dados para acelerar a exploração da nova linguagem sem alterar a submissão C++ atual.
